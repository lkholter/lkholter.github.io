Projects

Flashbar

Flashcards for Bartenders

Born from my own experience memorizing cocktail recipes, FlashBar is an app that helps bartenders study on the go. Because the internet was lacking an API for cocktails, I hard-coded in a few classics and users can build up their deck with their own recipes! And perhaps it's up to me to create that API.

Built using Ruby on Rails, interactivity with JavaScript. Also utilized HTML5, CSS3, SASS, Bootstrap, JQuery, and PostgreSQL.

Vagabond

A simple app for travel tips

This is a fun and simple group project that allows users to share travel tips with one another. I focused on creating the backend for this one, and learned a lot about style from a very talented team mate.

Built using Ruby on Rails, styled with Bootstrap4.

Found

A modern treasure hunt

Found is a web app based on the psychogeography philosophies of Guy Debord and the Situationists. It melds technological immediacy with physical experience. It's a reminder to engage with your environment and pay attention to the world. (It's a list of things you've found.)

Built using JavaScript, HTML5, CSS3 with NodeJS and Express. Authentication using Passport. MongoDB and Mongoose for database communication. Styled with Materialize.

Snake

The classic game

I love this game and had so much fun building it! The player uses the arrow keys to direct the hungry snake toward the mouse. Points are earned for every snack devoured but the game ends if the snake runs into a wall or its own body.

I built Snake using JavaScript, HTML5 Canvas, CSS3, and JQuery. Deployed using Firebase.
